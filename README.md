# Real Estate Analysis
Real Estate Analysis  This project demonstrates a simple data analysis workflow using SQL, Python, Pandas, and Excel.  The dataset contains information about real estate properties, including city, property type, price, area, and number of rooms.  Key analyses:  Average property price by city Number of properties by type Price per square meter  Tools used:  MySQL Python Pandas Excel GitHub
